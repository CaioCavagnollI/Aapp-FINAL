# Nexus Atlas

**A Plataforma Científica do Treinamento de Força — Powered by Atlas IA**

---

## O que é o Nexus Atlas?

Nexus Atlas é uma plataforma mobile (iOS/Android/Web) construída em React Native + Expo, focada em prescrição científica de treinamento de força. Reúne inteligência artificial, ferramentas acadêmicas, gestão de treino e uma loja de produtos educacionais em um único ecossistema.

---

## Instalação Rápida

```bash
# Clone o repositório
git clone <seu-repo>
cd nexus-atlas

# Instale dependências
npm install

# Configure variáveis de ambiente
cp .env.example .env
# Edite: DATABASE_URL, OPENAI_API_KEY, STRIPE_SECRET_KEY

# Inicie o backend
npm run server:dev

# Inicie o frontend (Expo)
npm run expo:dev
```

## Tecnologias

- **Frontend**: Expo 52 / React Native / Expo Router
- **Backend**: Node.js / Express / TypeScript
- **Banco de Dados**: PostgreSQL
- **IA**: OpenAI GPT-4 (streaming)
- **Pesquisa Científica**: PubMed, CrossRef, OpenAlex APIs
- **Pagamentos**: Stripe, Pix, PayPal
- **Fontes**: Outfit (Google Fonts)
